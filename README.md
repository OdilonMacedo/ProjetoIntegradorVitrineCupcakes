# ProjetoIntegradorVitrineCupcakes
Projeto Integrador 2 em Engenharia de Software - UP. 

# Objetivo
Desenvolver uma aplicacao web para uma cafeteria especializada em cupcakes, oferecendo uma vitrine virtual de produtos com funcionalidades de pedido, pagamento e acompanhamento de entrega.

# Situacao 1 Revisao e melhoria do planejamento
Nesta fase, foi realizada a revisao completa do planejamento, incluindo:
- Atualizacao dos diagramas UML (casos de uso, classes, sequencia e atividades);
- Revisao do prototipo de interface (IHC);
- Modelagem e documentacao do banco de dados;
- Organizacao de todos os arquivos no GitHub.

## Tecnologias previstas
- Front-end: HTML, CSS, JavaScript (View C#)
- Back-end: C# (.NET MVC)
- Banco de dados: SQL Server
