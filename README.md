# ProjetoIntegradorVitrineCupcakes
Projeto Integrador 2 em Engenharia de Software - UP. 

# Objetivo
Desenvolver uma aplicacao web para uma cafeteria especializada em cupcakes, oferecendo uma vitrine virtual de produtos com funcionalidades de pedido, pagamento e acompanhamento de entrega.

# Situacao 1 Revisao e melhoria do planejamento
Nesta fase, foi realizada a revisao completa do planejamento, incluindo:
- Atualizacao dos diagramas UML (casos de uso, classes, sequencia e atividades);
- Revisao do prototipo de interface (IHC);
- Modelagem e documentacao do banco de dados;
- Organizacao de todos os arquivos no GitHub.

# Situacao 2 Desenvolvimento da Solucao
Nesta etapa, foi desenvolvido o front-end e o back-end da aplicacao Vitrine de Cupcakes, conforme o planejamento definido na Situacao 1.
O objetivo é permitir que o cliente visualize os cupcakes, monte seu carrinho, finalize pedidos e acompanhe o status de entrega, garantindo uma experiência intuitiva e agradável

Tecnologias previstas
- Front-end: HTML, CSS, JavaScript (View C#), jQuery, Bootstrap
- Back-end: C# (.NET MVC), Entity Framework
- Banco de dados: SQL Server

Documentação e manuais
Foram elaborados os seguintes materiais complementares:

Manual do Usuário (PDF): explicando como navegar, comprar e acompanhar pedidos.
Demonstração em Video (.MP4): mostrando na prática o funcionamento da tela.

## Situação 3 – Testes e Validação

Participantes
Foram selecionados 5 colegas (dublês de usuários) para testar a aplicação.
Cada um executou os principais fluxos e registrou feedback em PDF.

Informações coletadas:

Nome e data do teste
Funcionalidades testadas
Resultados obtidos

Todos os feedbacks estão na pasta /feedbacks.

Resultados e melhorias aplicadas

Com base nos testes realizados, foram implementadas as seguintes correções:

Layout do carrinho quebrando no mobile: foi feito um ajuste completo da responsividade com Bootstrap
Mensagens de erro pouco claras: adicioando alertas visuais com textos mais visíveis
